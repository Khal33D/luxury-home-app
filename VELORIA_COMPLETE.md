## 🎬 Animation Details

### **Entrance Animations**
```
Initial: opacity: 0, y: 20-40px
Final:   opacity: 1, y: 0
Duration: 0.4-0.8s
Ease: easeOut
```

### **Hover Interactions**
```
Scale: 1 → 1.05 (products)
Y-axis: 0 → -8px (cards)
Opacity: 0 → 0.5 (overlays)
Duration: 0.3s
Transition: smooth
```

### **Scroll Reveals**
```
whileInView={{ opacity: 1, y: 0 }}
viewport={{ once: true }}
staggerChildren: 0.1-0.2s
Stagger delay animation across children
```

### **Particle Effects**
- 5-8 particles per section
- Random positioning
- Continuous floating animation
- Opacity pulsing
- Duration: 2-4s cycles
